## 运行说明

- 首先需要将链端的节点全部打开，参考官方文档中运行start_all.sh的方法。
- 接着就可以对项目进行构建了，在asset-app文件夹目录下，执行`./gradlew build`，然后进入dist文件夹，执行`bash asset_run.sh deploy`即可。
